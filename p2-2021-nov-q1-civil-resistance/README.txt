# Kula Revise interactive build

This folder is a deployable Kula Revise interactive.

Paper: DBE November 2021
Topic: Civil Resistance: UDF vs Botha's reforms
Questions: 21
Total marks: 50
Recommended time: 30 min (real exam: 60 min)

## Files in this folder
  index.html               - the page scaffold
  app.js                   - engine + paper data
  styles.css               - the Kula Revise stylesheet
  manifest.webmanifest     - PWA manifest
  sw.js                    - service worker (offline support)
  *.png / *.jpg / *.webp   - source images, if any

## Deploy
  Upload the entire folder to Vercel (or any static host).
  No build step needed; everything is plain static files.

## To regenerate after content edits
  Edit paper.json and re-run scripts/build.py.
